# AbsoluteLayout
Using an AbsoluteLayout requires the AbsoluteLayout module.
{%snippet absolute-layout-require%}

### Declaring an AbsoluteLayout
``` XML
<Page>
  <AbsoluteLayout>
    <Label text="This is Label 1" left="30" top="70" />
  </AbsoluteLayout>
</Page>
```
