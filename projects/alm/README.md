## Requirements

* Chrome
* NodeJS v6

## Usage

Get it:
```
npm install alm -g
```

Run it passing in the directory you want to serve up:
```
alm .
```

Now open it in `chrome` at the URL mentioned in your console. (**protip** use `alm -o` if your default browser is chrome)
