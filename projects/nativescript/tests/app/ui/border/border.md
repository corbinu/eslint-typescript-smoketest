# Border
Using borders requires the Border module.
{%snippet border-require%}

### Declaring a Border.
``` XML
 <Page>
     <Border borderRadius="10" borderWidth="2" borderColor="#FF0000">
         <Button text="OK"/>
     </Border>
 </Page>
```
